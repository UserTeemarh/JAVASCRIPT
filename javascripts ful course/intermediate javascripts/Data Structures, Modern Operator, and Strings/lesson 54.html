LOOPING ARRAYS: THE FOR-OF LOOP

The for-of statement is used to iterate over iterable objects such as arrays, strings, maps, sets, etc. It provides a cleaner syntax compared to traditional loops like for or while.

It executes a loop that operates on a sequence of values sourced from an iterable-object. Iterable objects include instances of built-ins, and other DOM collections and user defined iterables

LOOPING ARRAYS: THE FOR-OF LOOP SYNTAX

    • for (variable of iterable)
      statement;
  • Variable:
    Receives a value from the sequence on each iteration, May be either a declaration with const, let or var or assignment target (e.g a previously declared variable, an object property, or destructuring assignment pattern). Variables declared with var are not local to the loop, i.e. They are in the same scope the for-of loop is in
  • Iterable:
    An iterable object. The source of the sequence of values on which the loop operates.
  • Statement:
    A statement to be executed on every iteration. May reference variables. A block statement can be used to execute multiple statement.

LOOPING ARRAYS: THE FOR-OF LOOP DESCRIPTION

The for-of loop operates on the values sourced from an iterable on by one in sequential order. Each operation of the loop on a value is called an iteration, and on loop is said to iterate over the iterable. Each iteration executes statements that may refer to the current sequence value.

When for-of loop iterates over an iterable, it first calls the iterable's method, which returns an iterator, and then repeatedly calls the resulting iterator's method to produce the sequence of values to be assigned to variable.

The for-of loop exits when the iterator has completed.

The for-of loop exited early (e.g a break statement is encountered or an error is thrown), the return() method of the iterator is called to perform any cleanup.

  • for...of loop makes iterating through collections like arrays and strings straightforward, without the need to manage indexes manually.
  • However, to access indexes while using for...of, you can employ the .entries() method, which returns both the index and value of each item.

The entries() Method in for...of Loop

The entries() method is used to return an iterator object with key/value pairs of an array, when used in conjuction with the for-of loop, it iterates over both the index and the value of each element in the array.

  • SYNTAX:
    • const array = ['a', 'b', 'c'];

      for (const [index, value] of array.entries()) {
      console.log(index,value);
      }
