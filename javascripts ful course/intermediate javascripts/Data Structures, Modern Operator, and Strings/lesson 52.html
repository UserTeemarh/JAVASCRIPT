SHORT CIRCUTING (&& and ||) AND THE NULLISH COALESCING OPERATOR (??)

Short-circuiting is a behavior exhibited by logical operators (&&, ||) where the evaluation of the second operand is skipped if the outcome can be determined by evaluating the first operand alone.

In other words: Short Circuiting refers to the behavior where logical operators (&&, ||) don't evaluate their second operand if the result can be determined from the first operand alone.

The && Operator

The && operator performs a logical AND operation. When used in Short-circuiting, If the first operand evaluates to false, the entire expression evaluates to false without evaluating the second operand. This is because if one operand is false regardless of the value of the second operand. This behaviour is usesful for the conditional statements and can help improve performance by avoiding unnecessary evaluations.

  • The && Operator Syntax:
    • const value = 0;
      const result = value && 'Truthy Value';

      console.log(result);  // Output: 0

The || Operator:

The || operator performs a logical OR operation. When used in Short-circuiting, If the first operand evaluates to true, the entire expression evaluates to true without evaluating the second operand. This is because if one operand is true, the entire expression will always be true regardless of the value of the second operand.

  • The || Operator Syntax:
    • const name = '';
      const displayName = name || 'Guest';

      console.log(displayName); // Output: Guest;

The Nullish Coalescing Operator:

The nullish coalescing operator(??) is a relatively new addition to Javascript. It provides a way to provide a default value when accessing properties that may potentially be null or undefined.

How It Works:

  1. If the left operand is null or undefined, it returns the right operand.
  2. Otherwise, it returns the left operand.
  • The Nullish Coalescing Operator:
    • const foo = null ?? 'default string';
      console.log(foo); // Expected output: "default string"

      const baz = 0 ?? 42;
      console.log(baz);
      // Expected output: 0
