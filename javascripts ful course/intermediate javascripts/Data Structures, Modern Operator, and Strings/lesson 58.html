MAPS IN JAVASCRIPT

A map is a collection of key-value pairs. Unlike regular Javascript Objects, which only allow strings and symbols as keys, a Map allows keys of any type, including objects, functions, and primitives.

Key Features Of Map:

  • Any Type of Key: Any type of data type of value as a key can be used.
  • Ordered Entries: Entries in a Map maintain the order of their insertion.
  • Size Property: A map has a size property that returns the number of key-value pairs
  • Iterability: Map Objects are iterable, meaning iterable is allowed over their key-value pairs.
  • Creating a Map: A map can be created by using the new Map constructor
    • const myMap = new Map();
  • Adding Elements: To add elements to a Map, the set method is used.
    • myMap.set('Key', 'Value');
      myMap.set(1, 'Number')
      myMap.set(true, 'Boolean');
  • Accessing Elements: To access elements in a Map, the get method is used.
    • myMap.get('Key');
      myMap.get(1)
      myMap.get(true);
  • Checking for a Key: To check if a key exists in a Map, the has method is used.
    • myMap.has('Key');
      myMap.has(1)
      myMap.has(true);
  • Removing Elements: To remove an element from a Map, The delete method is used.
    • myMap.delete('Key');
  • Getting the size of a Map: To get the number of a key-value pairs in a Map, the size property is used.
    • myMap.size;
  • Iterating Over a Map: To iterate over a Map, for...of loop is used.
    • for (const [key, value] of myMap) {
      console.log(key, value) }
  • Converting a Map to an Array: To convert a Map to an array, the spread method is used.
    • const mapArray = [...myMap];
  • Clearing a Map: To remove all elements from a Map, the clear method can be used.
    • myMap.clear();

When to Use Map

  1. When a collection of key-value pairs is needed, where keys can be of any type.
  2. Developer need to maintain the insertion order of entries.
  3. To enquire efficient operations for adding, deleting, and querying key-value pairs.
